# Рабочий процесс: Генерация контента для Яндекс Дзен
## Ниша: Рассказы и истории / Семейные драмы

Каждый раз, когда пользователь называет **новую тему**, выполнять ВСЕ четыре задания по порядку:

---

## ЗАДАНИЕ 1 — Написать статью (≥20 минут чтения)

**Процесс (строго по ИНСТРУКЦИЯ.md):**

### ШАГ 1: Прочитать все файлы системы
- ИНСТРУКЦИЯ.md → СТИЛЬ.md → СТРУКТУРА.md → ИСТОЧНИКИ.md
- Читать полностью, не пропуская середину.

### ШАГ 2: Выбор структуры
Из СТРУКТУРА.md §1 выбрать подходящий шаблон:
- А — «Измена / Предательство»
- Б — «Конфликт поколений / Свекровь-тёща»
- В — «Прощение / Начать заново»
- Г — «Развод и дети»

Показать: какую структуру выбрал и почему.

### ШАГ 3: Таблицы ДО написания
Создать и показать все таблицы:
- Таблица 2.1 — Персонажи
- Таблица 2.2 — Хронология
- Таблица 2.3 — Фоновые факты (если нужны)

### ШАГ 4: Написание статьи
По СТИЛЬ.md + выбранной структуре + правилам монетизации Дзена.
Длительность чтения — не менее 20 минут (~6 000–9 000 слов).

### ШАГ 5: Верификация
- Таблица 2.4 — запрещённые клише
- Таблица 3.3 — чеклист Дзена и стиля
- Таблица 3.4 — проверка хронологии
- Таблица 3.5 — проверка повторов
- Таблицы 3.1, 3.2 — если есть факты/цитаты

**Формат вывода:**
```
## ВЫБОР СТРУКТУРЫ
## ТАБЛИЦЫ ДО НАПИСАНИЯ
## СТАТЬЯ
## ВЕРИФИКАЦИЯ
## СТАТУС: ✅ или ❌
```

---

## ЗАДАНИЕ 2 — Исправить ошибки (если найдены в верификации)

Если в верификации найдены ошибки (статус ❌):

1. **Анализ ошибок** — для каждой: какая слабость ИИ, какое правило нарушено
2. **Исправление** — переписать проблемные фрагменты
3. **Повторная верификация** — прогнать через те же таблицы

**Формат вывода:**
```
## АНАЛИЗ ОШИБОК
## ИСПРАВЛЕННАЯ СТАТЬЯ
## ПОВТОРНАЯ ВЕРИФИКАЦИЯ
## СТАТУС: ✅ или ❌
```

---

## ЗАДАНИЕ 3 — Сгенерировать 5 вариантов заголовков

**Требования:**
- Раскрывает суть — без кликбейта
- Привлекает внимание, создаёт любопытство
- Длина: 40–60 символов (максимум 80)
- Релевантен нише «Рассказы и истории / Семейные драмы»

**Формат вывода:**
```
1. [Заголовок] (X символов)
2. [Заголовок] (X символов)
3. [Заголовок] (X символов)
4. [Заголовок] (X символов)
5. [Заголовок] (X символов)

РЕКОМЕНДАЦИЯ: [номер], потому что [объяснение]
```

## ЗАДАНИЕ 4 — Промты для обложек (Stable Diffusion)

Эксперт по созданию обложек для Яндекс Дзен.

**Требования:**
- 5 вариантов промтов
- Персонажи — русской национальности
- Реалистичные фото, персонажи крупно и близко к экрану
- Сразу привлекают внимание аудитории
- Вариант А — для нейросети Stable Diffusion

**Формат вывода:**
```
Вариант А (Stable Diffusion):
[prompt на английском]

Вариант Б: [prompt]
Вариант В: [prompt]
Вариант Г: [prompt]
Вариант Д: [prompt]
```

## ВАЖНО

- Все четыре задания выполнять **последовательно** в одном ответе
- Актуальный год: **2026**
- Имена персонажей всегда анонимизировать (если история «из жизни»)
- Не публиковать при наличии хотя бы одного ❌ в верификации
- Файлы системы: ИНСТРУКЦИЯ.md + СТИЛЬ.md + СТРУКТУРА.md + ИСТОЧНИКИ.md
